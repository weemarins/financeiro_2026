# 🎉 BEM-VINDO!

## ⚡ Comece em 60 SEGUNDOS

### Linux/Mac
```bash
chmod +x scripts/setup.sh
./scripts/setup.sh

# Depois, em 2 terminais:
cd backend && npm run dev   # Terminal 1
cd frontend && npm run dev  # Terminal 2

# Abra: http://localhost:5173
# Email: joao@example.com | Senha: senha123
```

### Windows (PowerShell)
```powershell
# Executar em PowerShell como Administrador:
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope Process

# Depois:
node -v  # Verificar Node.js
npm install -g npm  # Atualizar npm

# Em 2 PowerShells:
cd backend; npm run dev    # PowerShell 1
cd frontend; npm run dev   # PowerShell 2

# Abra: http://localhost:5173
```

### Docker (Mais rápido)
```bash
docker-compose up -d
# Abra: http://localhost:5173
```
